Esse projeto foi feito para gerenciar meus muitos projetos iniciados
e incabados. A ferramenta foi feita para automatizar a criação de projetos
e acompanhar o seu desenvolvimento de forma automatica.

O projeto consiste em ser uma ferramenta para inicializar
de forma completa um projeto dado o seu tipo.

dex new --cpp projectName:
Cria o diretório projectName e o inicializa como diretório git
além disso, inclui os arquivos: Makefile, src/main.cpp, README.md, etc

dex new --py projectName:
Cria o diretório projectName e o inicializa como diretório git
além disso, inclui os arquivos: main.py, README.md

dex list:
Lista os projetos iniciados e informa características deles.
